binaryen.js

Easily create WebAssembly from JavaScript

Background: WebAssembly

A new binary executable format for the web. Compact and fast.

Background: Binaryen

A compiler infrastructure library for WebAssembly.

Helps build compilers to WebAssembly, and tools for processing WebAssembly.

Binaryen Optimizer

Reduces LLVM wasm binaries by ~6% (C++ and Rust)

Easy to do: 


wasm-opt input.wasm -Os -o output.wasm

Compiling with Binaryen

  • Provides tools to write a compiler to wasm (e.g., asm2wasm, s2wasm)
  • Has a C API
  • Has a JavaScript port: binaryen.js

binaryen.js

A JavaScript library that can run on the Web or Node.js, and lets you:

  • Create Binaryen IR
  • Optimize & minify it
  • Generate wasm from that

Code Sample


// if we ran enough iterations, return 0

module.if(
  module.i32.ge_u(                        // >= (unsigned)
    module.get_local(6, Binaryen.i32),    // the counter
    module.i32.const(MAX_ITERS_PER_PIXEL) // the limit
  ),
  module.return(
    module.i32.const(0)
  )
)

Creating wasm


// Create a module
var module = new Binaryen.Module();

// [..create a function type and the function body..]

// Create a function
module.addFunction('function_name', type, [], body);

// Optionally, optimize the module
module.optimize();

// Get a wasm binary (in typed array form)
var binary = module.emitBinary();

Using wasm


// Create an instance we can call
// (normally you'd use the async APIs, but this is tiny)
var wasm = new WebAssembly.Instance(
  new WebAssembly.Module(binary),
  {}
);

// Call it!
wasm.exports.function_name();
